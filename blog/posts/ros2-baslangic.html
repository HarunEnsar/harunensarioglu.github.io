ROS 2 ile Başlangıç Rehberi

2025-06-20 5 dk okuma Robotik

Robot Operating System (ROS), robotik yazılım geliştirme dünyasının en önemli araçlarından biridir. Bu yazıda ROS 2'ye giriş yapacağız.

>_ ROS 2 Nedir?

ROS 2, robotik uygulamalar için açık kaynaklı bir middleware framework'üdür. DDS (Data Distribution Service) üzerine kurulu olan ROS 2, gerçek zamanlı iletişim ve cross-platform desteği sağlar.

>_ Kurulum

Ubuntu 22.04 üzerinde ROS 2 Humble kurulumu:

sudo apt update && sudo apt install ros-humble-desktop
source /opt/ros/humble/setup.bash
echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc

>_ İlk Node

Python ile basit bir publisher node oluşturalım:

import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class MinimalPublisher(Node):
    def __init__(self):
        super().__init__('minimal_publisher')
        self.publisher_ = self.create_publisher(String, 'topic', 10)
        self.timer = self.create_timer(0.5, self.timer_callback)

    def timer_callback(self):
        msg = String()
        msg.data = 'Merhaba ROS 2!'
        self.publisher_.publish(msg)

def main():
    rclpy.init()
    node = MinimalPublisher()
    rclpy.spin(node)

if __name__ == '__main__':
    main()

Bu basit node, 0.5 saniyede bir topic kanalına mesaj yayınlar. Robotik projelerinizde sensör verisi, kontrol komutları ve daha fazlası için bu yapıyı genişletebilirsiniz.
