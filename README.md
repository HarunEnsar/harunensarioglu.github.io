# 🤖 harunensarioglu.github.io

Personal portfolio website — **Harun Ensarioğlu**, Robotic Software Developer.

## 🔗 Live Site
[https://harunensarioglu.github.io](https://harunensarioglu.github.io)

## 🛠️ Tech Stack
- **HTML5** — Semantic structure
- **Vanilla CSS3** — Design system with custom properties
- **Vanilla JavaScript** — Animations, interactions
- **GitHub Pages** — Static hosting

## 📂 Structure
```
├── index.html          # Main page (Hero, About, Skills, Experience, Projects, Contact)
├── cv/index.html       # CV page (TR/EN toggle, PDF embed)
├── blog/index.html     # Blog listing with category filters
├── blog/posts/         # Individual blog posts
├── assets/css/         # Design system, components, page styles
├── assets/js/          # Particles, main interactions
├── cv_en.tex           # LaTeX source (EN)
└── cv_tr.tex           # LaTeX source (TR)
```

## ✨ Features
- Dark robotics theme with neon cyan/purple accents
- Neural network particle animation (Canvas)
- Glassmorphism cards with hover glow effects
- Terminal-style UI elements
- Typing effect in hero section
- Scroll-triggered animations
- TR/EN language toggle
- Responsive design (mobile-first)
- Blog with category filtering
